#### 모범 답안 과 설명
##### 답안
정답은 lvalue입니다. </br>
const double PI = 3.14159; 코드에서 PI는 변수 이름으로, 변수는 값을 저장하는 lvalue입니다.

##### 설명
C++에서 lvalue는 다음과 같은 경우에 사용됩니다.
<li>변수</li>
<li>배열 요소</li>
<li>포인터</li>
<li>함수 이름</li>
<li>표현식의 왼쪽 피연산자</li><br>
