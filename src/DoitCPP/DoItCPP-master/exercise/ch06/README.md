이번 장에서는 객체지향 프로그래밍 개념과 4가지 특징인 추상화, 캡슐화, 상속성, 다형성에 관해 간단히 정리해 보았습니다. 그리고 4가지 특징을 C++로 표현할 수 있는 클래스도 알아봤습니다. 되새김 문제를 풀며 이번 장에서 배운 내용을 정리해 보세요.
